Welcome to Cizge.

Cizge is a work in progress but eventually i hope it to be an interactive "graph" calculator.

Syntax

Cizge has couple of operators. All of them are left associative

  • If there is no operator between vertex names they are simply pushed to the graph. Bracketed expressions with space in between acts like a list.
    • a b
    • ( a b )
  • Direction of the arrow denotes which vertex will be inserted to the other as child and returned for further evaluation. If there are no arrows, both of them inserted to each other and left hand side will be returned for further evaluation.
      This class of operators works like a cartesian product.
    • a "->" b
    • a "<-" b
    • a "--" b
      This class of operators works like zip.
    • "=>"
    • "<="
    • "=="
